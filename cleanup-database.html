🧹 Database Cleanup

Remove local image paths and migrate to Cloudinary

📌 What this does:

This tool will scan your database for events and users that have local file paths (like /uploads/image.jpg) and remove them. After cleanup, you'll need to re-upload images through the dashboard - they will automatically use Cloudinary.

⚠️ Important: Make sure you're logged in to your host account before clicking the button below.
